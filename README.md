# LevelUp - Test Dependencies Branch

This branch contains a test application to verify all dependencies are working correctly.

## What's Tested

- ✅ React Native + TypeScript
- ✅ Expo SDK ~54.0
- ✅ React Navigation (Stack + Bottom Tabs)
- ✅ Supabase Client
- ✅ React Native Paper (UI Library)
- ✅ React Native Reanimated (Animations)
- ✅ React Native Gesture Handler
- ✅ Expo Linear Gradient
- ✅ Vector Icons
- ✅ AsyncStorage
- ✅ Expo Secure Store

## Running the Test App

```bash
# Web
npm run web

# Android
npm run android

# iOS (Mac only)
npm run ios
```

## Features Demonstrated

- Linear gradient background
- Material Design cards
- Icons from multiple libraries
- Interactive animations (tap icon to test)
- Smooth scrolling
- TypeScript types working
